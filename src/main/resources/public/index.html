HTTP Dope

Various HTTP utils.

1. Echo HTTP

For mirroring HTTP requests:

1.1. Show My Request Headers

GET https://dope.alexn.org/echo/all
This also accepts an "ip" parameter, for overriding your IP: 
GET https://dope.alexn.org/echo/all?ip=185.210.218.104

1.2. Show My IP

GET https://dope.alexn.org/echo/ip

1.3. Geo IP

GET https://dope.alexn.org/echo/geoip
This also accepts an "ip" parameter, for overriding your IP: 
GET https://dope.alexn.org/echo/geoip?ip=185.210.218.104

1.4. Timeout

Endpoint for simulating a timeout. Example 1: 
GET https://dope.alexn.org/echo/timeout/3seconds
Example 2: 
GET https://dope.alexn.org/echo/timeout/1minute
The syntax for the given parameter is `<number><unit>`.
