HTTP Dope

Various HTTP utils.

1. Echo HTTP

For mirroring HTTP requests:

1.1. Show My Request Headers

GET @rootURL/echo/all
This also accepts an "ip" parameter, for overriding your IP: 
GET @rootURL/echo/all?ip=185.210.218.104

1.2. Show My IP

GET @rootURL/echo/ip

1.3. Geo IP

GET @rootURL/echo/geoip
This also accepts an "ip" parameter, for overriding your IP: 
GET @rootURL/echo/geoip?ip=185.210.218.104

1.4. Timeout

Endpoint for simulating a timeout. Example 1: 
GET @rootURL/echo/timeout/3seconds
Example 2: 
GET @rootURL/echo/timeout/1minute
The syntax for the given parameter is `<number><unit>`.

2. Vimeo

2.1. Config

Endpoint for fetching a video's configuration: 
GET @rootURL/vimeo/config/<uid>

Notes

Project was built out of personal necessity. If you'd like to inspect the source code or fork it, see:

github.com/alexandru/http-dope

Privacy Policy

No data is collected on these requests, nothing is stored, the code repository mentioned above gets deployed as is.
